Detailed Description

template<typename T, bool(*)(const T &, const T &) Orden = minimo<T>>
class Monton< T, Orden >

Plantilla de clase que implementa la estructura Montón (Heap) utilizando un arreglo dinámico. Permite configurarse como montículo mínimo (por default) o montículo máximo mediante un puntero a función.

Template Parameters
TTipo de dato que almacenará el montón.
OrdenPuntero a la función que determina si el montículo será mínimo o máximo.
